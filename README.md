# Task Manager Backend

## 📌 Overview

REST API backend for a task management system built with Spring Boot.

## 🛠 Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL

## 📂 Project Structure

* Entity: Database mapping
* Repository: JPA repository layer
* Service: Business logic
* Controller: REST API endpoints

## 🚀 Features

* User CRUD API
* Team / Position management
* Task management (planned)

## 🔗 API Example

* GET /users → 사용자 목록 조회
* POST /users → 사용자 등록
* PUT /users/{id} → 사용자 수정
* DELETE /users/{id} → 사용자 삭제

## ⚙️ Run

```
./gradlew bootRun
```
